# Data

This directory contains two sets of benchmark instances used in the computational experiments of the paper.

| Folder | Description | Source | Instances |
| :--- | :--- | :--- | :---: |
| `Biesinger/` | Small-scale instances from Biesinger et al. (2016) | [Biesinger et al. (2016)](https://www.ac.tuwien.ac.at/research/problem-instances) | 1 |
| `Qi/` | Large-scale instances generated by the authors following the procedure of Qi et al. (2024) | This paper, Section 7.2 | 9 |

---

## References

> **Biesinger B, Hu B, Raidl G (2016)** Models and algorithms for competitive facility location problems with different customer behavior. Ann. Math. Artif. Intell. 76(1-2):93–119.

> **Qi M, Jiang R, Shen S (2024)** Sequential competitive facility location: Exact and approximate algorithms. Oper. Res. 72(1):300–316.
